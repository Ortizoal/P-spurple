# P's Purple - Makeup & Hairstyling Website

A modern, fully responsive website for P's Purple beauty business, offering makeup and hairstyling services with appointment booking, pricing, and admin insights.

## Features

### 🎨 **Services & Pricing**
- View makeup and hairstyling service packages
- Detailed rate card with pricing
- Individual package details

### 📅 **Appointment Management**
- Book appointments online
- View calendar of appointments
- Manage bookings

### 📊 **Admin Dashboard**
- Business insights and analytics
- Revenue tracking
- Appointment metrics

### 📱 **Responsive Design**
- Fully responsive on mobile, tablet, and desktop
- Touch-friendly interface
- Optimized performance

## Color Scheme
- **Primary**: Purple (#6B4C8A)
- **Secondary**: Deep Purple (#4B2E64)
- **Accent**: Gold (#D4A574)
- **Background**: Light Purple (#F3E5F5)
- **Text**: Dark Gray (#333333)

## Getting Started

1. Open `index.html` in any web browser
2. Navigate using the menu
3. No installation or build process needed

## Browser Support
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Pages

### Home (`index.html`)
Landing page with business introduction, service overview, and featured packages.

### Rate Card (`rate-card.html`)
Detailed pricing for all makeup and hairstyling services with descriptions.

### Appointments (`appointments.html`)
Client booking system to schedule services.

### Calendar (`calendar.html`)
Visual calendar showing all booked appointments.

### Admin Dashboard (`insights.html`)
Business analytics, revenue reports, and booking metrics.

## Customization

### Update Business Info
Edit the business information in each HTML file to match your details.

### Add Images
Replace placeholder images in the `assets` folder with your actual business photos.

### Modify Services
Update service offerings and pricing in `rate-card.html` and `appointments.html`.

### Color Changes
Modify the CSS variables in `css/styles.css` to change the color scheme.

## Storage
- Appointment data is stored in browser localStorage
- Data persists between sessions
- Clear browser cache to reset data
