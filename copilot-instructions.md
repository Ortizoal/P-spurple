# P's Purple Website - Project Instructions

This is a professional, responsive website for P's Purple makeup and hairstyling business.

## Project Overview
- **Website Type**: Responsive Business Website
- **Framework**: HTML5, CSS3, Vanilla JavaScript
- **Color Scheme**: Purple (#6B4C8A) with complementary background colors
- **Target**: Mobile and Desktop Responsive

## Features
1. **Home Page** - Service showcase and business introduction
2. **Rate Card** - Pricing for makeup and hairstyling services
3. **Appointment Booking** - Client scheduling system
4. **Calendar** - Visual appointment calendar
5. **Admin Insights Dashboard** - Business analytics and metrics
